\section*{Project Ideas}

Select \textbf{one} of the following projects, or propose your own (both of which need to be approved via the project proposal).

\begin{itemize}
    \item \textbf{Least Squares FIR Filter Design:} Design an arbitrary linear phase FIR filter using least squares method. Include at least 40 dB of range in your target magnitude response. Demonstrate target response versus actual response.
    
    \item \textbf{Sigma-Delta ADC Simulation:} Take a sampled signal, upsample it (like HW 5), then quantize with a 1-bit Sigma-Delta ADC model. Filter and downsample the 1-bit output back to the original rate. Compare to original signal. How many bits resolution do you have?
    
    \item \textbf{Adaptive Filter Implementation:} Implement adaptive filtering for noise cancellation.
    
    \item \textbf{Audio Effects Processor:} Design and implement digital audio effects such as reverb, pitch shifting, or parametric equalization.
    
    \item \textbf{Multirate Filter Bank:} Design and analyze a multirate filter bank system for signal decomposition and reconstruction.
    
    \item \textbf{Speech Analysis System:} Implement speech processing techniques such as formant analysis, pitch detection, or compression.
    
    \item \textbf{Image Processing Application:} Apply 2D filtering techniques for edge detection, noise reduction, or feature enhancement.
    
    \item \textbf{Your Own Idea:} Propose a DSP project related to course topics. Submit a proposal for approval before December 10.
\end{itemize}

\section*{Report Structure}

Your report should include the following sections:

\begin{enumerate}
    \item \textbf{Introduction:} Brief overview of your chosen project and objectives
    
    \item \textbf{Theory:} Relevant equations and DSP concepts used in your design
    
    \item \textbf{Implementation:} Description of your code and design choices
    
    \item \textbf{Results:} Plots, measurements, and analysis of system performance
    \begin{itemize}
        \item Include properly labeled figures with captions
        \item Show frequency responses, time-domain signals, or spectrograms as appropriate
        \item Provide quantitative performance metrics
    \end{itemize}
    
    \item \textbf{Discussion:} Interpretation of results, design tradeoffs, and challenges encountered
    
    \item \textbf{Conclusion:} Summary of findings and potential improvements
    
    \item \textbf{Appendix:} Complete code.
\end{enumerate}

\section*{Evaluation Criteria}
